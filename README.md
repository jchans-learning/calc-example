# 作業：計算機功能的網頁

## Spec

1. 計算機裡的功能 +, -, \*, /, =, clear(C)
2. 畫面上會有兩個 readonly input box, 一個可以及時顯示「算式」 ，另一個按下「 = 」功能鍵後可以顯示「結果」。
3. 數字計算上限為 2^32，超過請顯示超過數字上限
4. 手機版畫面要求: 寬度為畫面 100%，高度最大為畫面 50%，若高度超過顯示範圍則可 scroll
5. 電腦版畫面要求: 計算機可以被拖拉至畫面任一位置，是否可放到超出螢幕範圍 candidate 自行決定
